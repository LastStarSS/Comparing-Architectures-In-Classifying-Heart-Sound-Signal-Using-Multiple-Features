# How to use
1. Clone the repository
2. Go to "Code" folder
```
cd Code
```
3. Run any of the Python files
```
python compare.py
```

# Files
1. original.py
- Original MATLAB code converted to Python
2. optimized.py
- Optimized version of original.py
- Uses function-main structure
3. revised.py
- Improved version of optimized.py
- Added arguments
- Improved model structure, including early stopping
4. compare.py
- Compare the CNN model in revised.py to LSTM and RCNN
- Added plots to compare accuracy and loss of the models
